Check if the first interval is lexicographically smaller than the second.

Compare two intervals lexicographically. Note that this is not a partial order of intervals in the mathematical sense, e.g., if [a, b] < [c, d], you cannot infer that b < c. Instead, it is a total order used to put Intervals into containers.

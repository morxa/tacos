Detailed Description

template<typename ValueT, typename NodeT>
class tacos::search::Heuristic< ValueT, NodeT >

The heuristics interface.

A heuristic is used by the search to determine which open node to expand next. This class needs to be specialized to actually implement a heuristic function.

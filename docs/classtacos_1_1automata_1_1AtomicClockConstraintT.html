Detailed Description

template<class Comp>
class tacos::automata::AtomicClockConstraintT< Comp >

An atomic clock constraint.

This is a templated atomic constraint, where the template parameter is the comparison operator, e.g., to define a constraint x <= 3 on a clock x, use AtomicClockConstraintT<std::less_equal>(3).

Template Parameters
Compthe comparison operator, e.g., std::less

template<typename LocationT >
using tacos::automata::ata::Configuration = typedef std::set<State<LocationT> >

An ATA configuration is a set of ATA states.

An ATA configuration is defined by a set of states, which in turn are pairs of an ATA location and a clock valuation of the ATA's single clock.

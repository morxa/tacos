Detailed Description

template<typename ValueT, typename NodeT>
class tacos::search::TimeHeuristic< ValueT, NodeT >

The Time heuristic, which prefers early actions.

This heuristic computes the accumulated time from the root node to the current node and prioritizes nodes that occur early.

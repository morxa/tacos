Detailed Description

template<typename ValueT, typename NodeT, typename ActionT>
class tacos::search::PreferEnvironmentActionHeuristic< ValueT, NodeT, ActionT >

Prefer environment actions over controller actions.

This heuristic assigns a cost of 0 to every node that has at least one environment action as incoming action. Otherwise, it assigns the cost 1.

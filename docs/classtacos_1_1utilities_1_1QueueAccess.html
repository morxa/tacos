Detailed Description

template<class Priority, class T>
class tacos::utilities::QueueAccess< Priority, T >

Get direct access to the job of a thread pool.

The ThreadPool must not be running, i.e., you should use either the ThreadPool's workers, or start the ThreadPool with StartOnInit::NO and access the queue manually. Direct queue access is mainly helpful for testing and for single-threaded, synchronous queue processing.

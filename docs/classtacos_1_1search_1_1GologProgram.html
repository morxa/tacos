Detailed Description

Wrapper for a Golog++ program.

This class manages a Golog++ program and provides additional functionality needed for synthesizing a controller against this program.

all_action_clocksIf set to true, insert a clock for every action.
action_clock_namesA set of actions for which to insert a clock. Actions may be start or end actions, the action names should contain "start" or "end" for durative actions.
